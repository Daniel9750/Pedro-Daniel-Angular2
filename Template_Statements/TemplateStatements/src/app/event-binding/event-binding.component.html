Template statements

Template statements are methods or properties used to respond to user events.

Clicking on the orange button below will both increment the input's value by one and print a like emoji (👍🏻) down the button, using template statements.

likesNumber attribute value (From the template) => {{ likesNumber.value }}

likesNumber property value (From the component) => {{ likesNumber }}

❗Angular prioritizes template context over component one when there's a name conflict ("likesNumber" in this case)❗
